Obrik v0.1

You don't seem to have Java installed.

This is a very limited applet. If it were more advanced, there would be a larger number than "0.1" after the "v" up top.

Instructions:

You can draw only one thing: straight lines. Lines, when put together, will be interpreted as one of 2 things:

  1. Triangles: these will be turned into physical bodies.
  2. X's: When drawn over triangles, the triangles won't move.
You will probably want to draw a triangle or two, then hit the "Run" button. When you're done watching the simulation, hit "Reset" and either play it again, or alter what you've drawn.

I expect you to do the following:

  1. Draw lines to create triangles.
  2. Draw X's over some of the triangles.
  3. Hit the "Run" button and watch.
  4. Hit the "Reset" button when you're done.
To reset the state of Obrik, refresh your browser window.
